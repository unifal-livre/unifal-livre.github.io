Metodologia

O que é uma metodologia?

Uma metodologia é uma maneira racional de se realizar determinado processo repetidas vezes. No nosso caso, uma maneira racional de realizarmos nossas pesquisas de maneira a causar credibilidade.

Como é desenvolvida uma metodologia?

Uma metodologia séria é desenvolvida observando-se todos os processos necessários para que se atinja o que se deseja de maneira eficiente.

Como a metodologia desenvolvida deve ser disponibilizada?

A metodologia deve ser pública caso se tenha interesse em resultados públicos, para que qualquer um verifique a autenticidade dos resultados e entenda como se chegou neles.

Como é a nossa metodologia científica?


a) Identificação do problema;

b) Coleta da fonte de dados;

c) Coleta dos dados;

d) Desenvlvimento de metologia específica do estudo;

e) Aplicação da metologia específica;

f) Checagem dos dados obtidos e da metologia específica adotada;

g) Resumo das informações;

h) Escrita do texto-resumo e geração dos arquivos a serem disponibilizados;
